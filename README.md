# Evolução dos Animes Shonen

**Cristyan Vessimo Gomes de Oliveira**

## Descrição da Atividade

Projeto de site com 4 páginas desenvolvido em HTML5 e CSS3 sobre a **evolução dos animes shonen**,
abordando três obras fundamentais: Cavaleiros do Zodíaco, Dragon Ball Z e Bleach.

## Páginas

| Arquivo | Conteúdo |
|---|---|
| `index.html` | Apresentação geral + cards + lista sobre shonen |
| `cdz.html` | Cavaleiros do Zodíaco — história e personagens |
| `dbz.html` | Dragon Ball Z — sagas + vídeo do YouTube |
| `bleach.html` | Bleach — arcos + formulário com 7 controles |
